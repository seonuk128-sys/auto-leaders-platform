# 오토리더스(Auto Leaders) 상담 신청 플랫폼

본 프로젝트는 장기렌트/리스 및 보험 영업을 위한 전문 상담 신청 랜딩 페이지 구축 프로젝트입니다.

## 📝 프로젝트 개요 및 목표
- **목표:** 고객 맞춤형 랜딩 페이지 제공을 통한 DB 수집 효율 극대화
- **핵심 가치:** 신뢰감 있는 브랜딩, 직관적인 UI/UX, 실시간 알림 시스템

## 🛠️ 주요 구현 기능

### 1. 전용 랜딩 페이지 (라우팅)
- `/rent`: **장기렌트/리스 전용** 상담 신청 페이지
- `/insurance`: **보험 영업 전용** 상담 신청 페이지

### 2. 장기렌트 폼 고도화 (이미지 분석 기반)
- **카드형 선택 시스템:** 이용상품(렌트/리스), 계약기간(1~5년), 선납금, 주행거리 등을 카드 형태로 선택
- **툴팁 도움말:** 각 항목 옆 `?` 아이콘 클릭 시 상세 설명 제공
- **최다 배지:** 가장 선호하는 조건에 주황색 `최다` 배지 표시로 신뢰감 부여
- **브랜딩:** 오토리더스(Auto Leaders) 로고 및 맞춤형 타이포그래피 적용

### 3. 알림 및 관리 시스템
- **실시간 이메일 알림:** 신청 즉시 Gmail을 통해 고객 상세 정보 및 상담 조건 발송
- **데이터베이스 저장:** SQLite3를 활용하여 모든 상담 내역을 안전하게 기록
- **API 연동:** Express 서버를 통한 통합 상담 신청 API(`POST /api/consultations`) 구현

## 🚀 실행 방법 (Local)

### 1. 서버 실행 (Port: 5001)
```bash
cd server
node index.js
```

### 2. 프론트엔드 실행 (Port: 5173)
```bash
cd client
npm run dev
```

## 🌐 향후 배포 계획 (Deployment)
- **방법:** Render(렌더) 플랫폼을 활용한 무료 정적/동적 웹 배포
- **전략:** 
  - 이메일 실시간 수신을 핵심 경로로 설정 (무료 플랜의 데이터 초기화 영향 무관)
  - `https://` 보안 연결 자동 적용
  - 추후 개인 도메인(`autoleaders.kr` 등) 연결 가능
